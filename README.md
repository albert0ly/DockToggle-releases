# 🎯 DockToggle

> **Windows-style taskbar behavior for your Mac** — Click any Dock icon to instantly hide or show that app.

## About DockToggle

DockToggle is a lightweight macOS menu bar utility that transforms how you interact with your applications. Tired of clicking an app's Dock icon only to have it come to the foreground? DockToggle brings familiar Windows-style taskbar behavior to Mac, letting you toggle app visibility with a single click.

### What It Does

- **Click to Hide/Show**: Click any Dock icon to instantly toggle that app's visibility
- **Menu Bar Integration**: Runs quietly in your macOS menu bar
- **Always Available**: Quick access from your menu bar to toggle DockToggle on/off
- **Zero Configuration**: Works out of the box — just install and enjoy
- **Lightweight**: Minimal system resources, runs in the background

### Why You'll Love It

If you're switching from Windows to Mac, or simply prefer a more efficient workflow, DockToggle restores the productivity boost of Windows-style window management. Hide distracting applications instantly. Switch focus faster. Stay in the zone.

## Installation

### From GitHub Releases (Recommended)

1. Download the latest release from the [Releases page](https://github.com/albert0ly/DockToggle-releases/releases)
2. Extract the **DockToggle.zip** file
3. Drag **DockToggle.app** to your **Applications** folder
4. Open **Applications** → **DockToggle** to launch it
5. Grant any requested permissions in System Settings

### First Launch

On your first launch, macOS may show a security warning since DockToggle is not notarized by Apple. This is normal for independently developed applications:

- **Right-click** DockToggle.app and select **Open**
- Confirm the security prompt
- DockToggle will run and a menu bar icon will appear

To remove quarantine attributes (optional):
```bash
xattr -dr com.apple.quarantine ~/Applications/DockToggle.app
```

### Launch at Startup (Optional)

DockToggle includes an option to launch automatically when you log in. You can toggle this in the DockToggle menu bar menu.
