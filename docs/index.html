Features

How it works

1

Download & open

Drag DockToggle to Applications, then open it. macOS will ask you to allow Accessibility access.

2

Grant Accessibility permission

DockToggle needs Accessibility access to intercept Dock clicks. Grant it in System Settings → Privacy & Security → Accessibility.

3

Done

Click any running app's Dock icon while it's focused — it hides. Click again — it comes back.

FAQ

Why does macOS ask me to re-grant Accessibility permission after an update?

macOS tracks permission grants by the app's binary signature. Because DockToggle is distributed without an Apple Developer certificate, macOS may treat a new version as a different app. If DockToggle already appears in your Accessibility list, toggle the switch off and back on — you won't need to remove and re-add it.

Is DockToggle safe to use?

Yes. DockToggle is unsigned, which means macOS shows a warning on first launch — this is standard for apps not distributed through the App Store or by a registered developer. The app makes no network connections and accesses only what the Accessibility permission grants.

Why does it need Accessibility permission?

The Accessibility API is the only supported way for a macOS app to observe and intercept Dock icon click events. DockToggle uses it solely for this purpose.

Does it work on Apple Silicon?

Yes. DockToggle is a universal binary — it runs natively on both M-series (Apple Silicon) and Intel Macs.

What macOS versions are supported?

macOS 12 Monterey and later.
